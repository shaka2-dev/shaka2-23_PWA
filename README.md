# 🪵 Shaka2瞑想

**NHK時報スタイル・心身調律歯磨きアプリ**

---

## 📖 プロジェクト概要

**Shaka2瞑想** は、鳥井式心身調律歯磨き法に基づいた、NHK時報のリズムを用いた瞑想的な歯磨きアプリです。

木魚の音と調和呼吸のビジュアルフィードバックにより、ユーザーのペースに合わせた心身調律を実現します。モバイル向け PWA (Progressive Web App) として、オンライン・オフライン両対応です。

---

## ✨ 主な機能

### 🪥 **3つの磨きモード**

| モード | 説明 | 対象 |
|--------|------|------|
| **エリア磨き** | 全体の基本 | 24エリア |
| **歯間磨き** | 隣り合う歯 | 複数歯間 |
| **ワンタフト磨き** | 細かい部分 | 個別スポット |

### 🧠 **ブレイン機能**

利き手と非優位手を自動切り替え。脳の活性化を促進します。

### 🦷 **欠損歯管理**

欠損歯をタップで登録。自動的にスキップされます。

### 🎵 **NHK時報音**

助走として使用。60秒カウントダウン後、自動的に木魚リズム開始。

### 📹 **カメラ統合**

背景にリアルタイムカメラ映像を表示（許可時）。

### 💾 **ローカルストレージ**

すべての設定を自動保存。オフライン対応。

### 📱 **PWA対応**

- ホーム画面にインストール可能
- オフライン動作
- 低速ネットワーク対応
- Service Worker キャッシング
