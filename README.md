# 🪵 Shaka2瞑想（修正版23・PWA版）

## 📖 プロジェクト概要

**Shaka2瞑想** は、NHK時報スタイルの **心身調律歯磨きアプリ** です。

- 🎵 鳥井式心身調律歯磨き法に基づいた設計
- 📱 モバイル向け PWA（Progressive Web App）対応
- 🌐 GitHub Pages でデプロイ
- 🎨 ダークモード UI

### 公開URL
**https://shaka2.dev.github.io/shaka2-23_PWA/**

---

## ✨ 主な特徴

### ✅ 機能
- **3つの磨きモード**
  - 🪥 **エリア磨き** … 全体の基本磨き
  - 🔗 **歯間磨き** … 歯と歯の間
  - 🔍 **ワンタフト磨き** … 細かい部分

- **複数の磨き方**
  - 標準モード … 利き手1回
  - 念入りモード … 利き手2回
  - ブレインモード … 利き手＋非優位手

- **NHK時報スタイルの音**
  - 助走：ポ→ポ→ポ→ポーン（4秒）
  - 木魚のリズム音（1秒ごと）
  - 段階的な吸・吐ビジュアル表示

- **欠損歯管理**
  - 欠損歯を事前登録
  - 自動的に対象から除外

- **カメラミラー機能**
  - 磨いている様子をライブ映像で確認
  - Android / iPhone / PC すべて対応

### 🔧 技術仕様
- **フレームワーク** … バニラ JavaScript（フレームワーク不使用）
- **プラットフォーム** … PWA（Progressive Web App）
- **キャッシング** … Service Worker v8
- **マニフェスト** … manifest.json
- **デプロイ** … GitHub Pages（HTTPS）
