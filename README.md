## 概要

### 🎯 目的

自分のペースで、利き手と非優位手を活用しながら系統的に歯を磨くアプリケーション。
音声ガイダンス（木魚の音）に従いながら、瞑想的な歯磨きを実現します。

### 📱 対応環境

- **PC**: Windows, Mac, Linux（Chrome, Edge, Safari推奨）
- **スマートフォン**: Android, iOS（Chrome推奨）
- **オフライン対応**: ✅ 可能（Service Worker 対応）
